
# Summary v1.0

## Api Rules

- url starts with 'api'
    
    e.g. `xxxx/api/articles`

- response contains two parts: `code` and `res`

    the `code` status mapping is [here](https://demon16.github.io/)
    
## Main Body

Article apis

  - `/articles`
  - `/articles/article_id`

Game apis

  - `/games`
  - `/games/game_id`
